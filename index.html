How To Play

Try your best to get the right answer after solving the equation within the time limit until the game reaches the end.

The only operand/number is number 1 and the operators/operations are addition(+), substraction(-), multiplication(*) and division(/). The result can be Infinity or Underfined if there's a zero division.

You can play the game using keyboard. All the number key from 0 to 9 have been binded. For INF/UND button, the corresponding key would be the key to the right of the key 0 on the keyboard. (Key '-/_' on the horizontal line or Key "./Del" on numpad).

Equation Evaluation

For each equation, the final result is computed in the following ways:

  1. Use absolute function abs() to make negative number into positive number.
  2. If the new result is greater then 10, subtract it by 10.

Example:
Result = -0 (1)→ 0 (2)→ 0
Result = -4 (1)→ 4 (2)→ 4
Result = 11 (1)→ 11 (2)→ 1
